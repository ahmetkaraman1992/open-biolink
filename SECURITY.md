# Security Policy

## Reporting a vulnerability

If you discover a security issue in Open BioLink, please report it privately before creating a public issue.

You can contact the maintainer through GitHub.

## Scope

Security reports may include:

- Unsafe link handling
- Dependency vulnerabilities
- Cross-site scripting risks
- Misconfigured deployment examples
- Privacy or tracking concerns

## Maintainer response

The maintainer will review valid reports and prioritize fixes based on severity and impact.
