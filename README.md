# Open BioLink

Open BioLink is an open-source, self-hostable bio link page template for creators, freelancers, indie hackers, and small businesses.

It helps users publish a fast, customizable, privacy-friendly link-in-bio page without depending on closed bio-link platforms.

## Why this project exists

Most bio-link platforms are closed, subscription-based, and limited in customization. Open BioLink gives individuals and small teams a simple open-source alternative they can host, customize, and extend on their own.

## Features

- Self-hostable link-in-bio page
- Config-based profile and links
- Responsive mobile-first design
- Dark/light theme support
- Social links and custom CTA buttons
- Easy deployment to Vercel, Netlify, or static hosting
- MIT licensed

## Use cases

- Personal bio link page
- Creator landing page
- Freelancer contact page
- Indie hacker product links
- Small business social/contact hub

## Getting Started

```bash
npm install
npm run dev
```

## Configuration

Copy `config.example.json` and update your profile, links, theme, and social accounts.
